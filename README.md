react-beaker
===

### Install

```
npm install -g wizawu/react-beaker
```

### Usage

1. Code struct must be like

    ```
    path/to/source/
    ├── html
    ├── js
    │   └── entries
    └── package.json (optional)
    ```

2. Commands

    ```
    react-beaker watch path/to/source
    react-beaker build path/to/source
    react-beaker publish path/to/source
    ```
